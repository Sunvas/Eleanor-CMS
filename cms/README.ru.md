В этом каталоге находятся внутренние файлы Eleanor CMS.

Здесь размещаются ядро CMS, конфигурация, кэш, логи, юниты, шаблоны, виджеты, файлы локализации, рабочая область Composer и другие файлы, необходимые для работы CMS.

Содержимое этого каталога не должно отдаваться web-сервером напрямую. Доступ к каталогу из браузера должен быть закрыт на уровне web-сервера. Публичные статические файлы должны размещаться в каталоге `static/`.
