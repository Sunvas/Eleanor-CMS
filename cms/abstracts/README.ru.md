# Каталог абстрактных PHP-классов

Данный каталог предназначен для размещения абстрактных PHP-классов, используемых в Eleanor CMS.

## Принцип работы автозагрузчика

Автозагрузчик обслуживает только пространство имён `CMS\Abstracts`. При запросе класса автозагрузчик преобразует имя класса в имя файла, приводя его к нижнему регистру и дополнительно проверяя kebab-case вариант имени. Например, запрос:

```php
CMS\Abstracts\TestClass
```

приведёт к последовательной проверке файлов:

```text
testclass.php
test-class.php
```

Поддерживается организация классов через подкаталоги. Каждый подкаталог соответствует дополнительному уровню пространства имён внутри `CMS\Abstracts`. Например, для класса:

```php
CMS\Abstracts\Test\SomeFeature
```

автозагрузчик выполнит поиск файлов:

```text
./test/somefeature.php
./test/some-feature.php
```

---

## Правила и требования

### 1. Именование файлов

Автозагрузчик поддерживает оба варианта имён файлов:

```text
testclass.php
test-class.php
```

Имена файлов регистрозависимы и должны быть записаны в нижнем регистре.

### 2. Независимость от namespace и имени класса

Автозагрузчик определяет соответствие класса исключительно по имени файла. Фактическое пространство имён и имя класса внутри файла не участвуют в поиске и могут быть произвольными. Благодаря этому каталог может использоваться как адаптерный слой для сторонних библиотек или внешних абстрактных классов.

### 3. Возврат полного имени класса (FQCN)

Если класс внутри файла не соответствует ожидаемому имени класса в `CMS\Abstracts`, файл должен вернуть полное квалифицированное имя класса (FQCN):

```php
return \Vendor\Package\ClassName::class;
```

Пример:

```php
<?php
namespace Vendor\Package;

abstract class ClassName
{
    // ...
}

return \Vendor\Package\ClassName::class;
```

После загрузки файла автозагрузчик:
- использует класс напрямую, если он уже принадлежит пространству имён `CMS\Abstracts`;
- либо автоматически создаёт псевдоним внутри `CMS\Abstracts`.

### 4. Автоматическое создание псевдонима

Класс не обязан принадлежать пространству имён `CMS\Abstracts`. Если файл возвращает FQCN класса из другого пространства имён, автозагрузчик автоматически создаёт псевдоним внутри `CMS\Abstracts`, обеспечивая доступ к классу через единый механизм автозагрузки. Например:

```php
CMS\Abstracts\Test\SomeFeature
```

может фактически ссылаться на:

```php
\External\Library\FeatureClass
```

### 5. Необязательный `return`

Если класс уже объявлен в ожидаемом пространстве имён и его имя соответствует имени файла, файл может не возвращать FQCN. Например, файл:

```text
test/some-feature.php
```

может напрямую содержать:

```php
<?php
namespace CMS\Abstracts\Test;

abstract class SomeFeature
{
    // ...
}
```

В этом случае автозагрузчик обработает класс автоматически без дополнительного `return`.
