В этом каталоге находятся шаблоны панели администратора.

Файлы шаблонов — это PHP-файлы, которые подготавливают и выводят HTML-разметку панели администратора. Они могут использовать переменные CMS, файлы локализации, общие include-шаблоны и контекстные данные, переданные admin-юнитами.

Рекомендуемая структура:

- `index.php` — основной layout панели администратора.
- `SignIn.php` — шаблон страницы входа.
- `error.php` — шаблон страницы ошибки.
- `app.php` — шаблон-обёртка для Vue-приложений.
- `includes/` — переиспользуемые фрагменты шаблонов.
- `l10n/` — файлы локализации общих шаблонов панели администратора.
- `sidebar/` — шаблоны элементов бокового меню.
- `unit_name/` — шаблоны конкретного юнита в панели администратора.
