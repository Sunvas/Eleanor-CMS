This directory contains Eleanor CMS configuration files.

Configuration files can be stored in two formats:

1. `.json` files are generated by the system and loaded first.
2. `.php` files are loaded after JSON files and may override or extend their settings. Inside a PHP configuration file, the `$config` array contains values loaded from the corresponding JSON file.

PHP configuration files are used when comments are needed or when configuration values must be calculated dynamically instead of being stored as static JSON data.

In monolingual mode, localized values are stored as strings. In multilingual mode, localized values must be stored as arrays where keys are language codes and values are localized strings:

```php
[
	'en'=>'My site',
	'ru'=>'Мой сайт',
]
```
