[English](README.md)

В этом каталоге находятся прямые endpoint-файлы пользовательской части.

Если slug запроса соответствует `^[a-z\d\-_.]+$` и не был обработан юнитом или статической страницей, Eleanor CMS подключает одноимённый PHP-файл из этого каталога.

Например, запрос `/demo-json` подключит файл `cms/direct/demo-json.php`.

Прямые файлы удобны для простых endpoint-ов, которым не нужен полноценный юнит: plain text-ответов, JSON-ответов, callback-ов или небольших самостоятельных страниц.

В этом каталоге также находятся демонстрационные файлы с разными типами ответа:

- `demo-direct.php` — HTML-страница, отрендеренная через шаблон CMS.
- `demo-json.php` — JSON-ответ.
- `demo-text.php` — plain text-ответ.
