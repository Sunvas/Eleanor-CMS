This directory stores diagnostic logs generated by Eleanor PHP Library.

The log writer uses three file types:

- `.log` — human-readable log entries.
- `.json` — structured data used to group and update identical log entries.
- `.lock` — lock files used to protect logs from concurrent writes.

Logs may include CMS exceptions, system errors, data errors, user errors, and MySQL-related errors. MySQL errors are grouped separately by type, such as connection, query, prepared statement, value, and other database errors.

Oversized `.log` files are compressed to save disk space. Their related `.json` grouping data is removed during compression because oversized logs are assumed to be inactive. Logs are not deleted automatically.
