### English
This directory is designated for storing website logs.
CMS writes here information about PHP exceptions for debugging purposes. The system archives old logs to save space, but they are retained indefinitely.

### Русский язык
Это каталог отведён для хранения логов сайта.
Система записывает сюда информацию о PHP исключениях, которые требуют исправления. Для экономии места, система архивирует логи, но автоматически они не удаляются.
