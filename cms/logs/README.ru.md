[English](README.md)

В этом каталоге хранятся диагностические логи, созданные библиотекой Eleanor PHP Library.

Механизм логирования использует три типа файлов:

- `.log` — человекочитаемые записи логов.
- `.json` — структурированные данные для группировки и обновления одинаковых записей.
- `.lock` — lock-файлы для защиты логов от одновременной записи.

В логах могут быть исключения CMS, системные ошибки, ошибки данных, пользовательские ошибки и ошибки MySQL. Ошибки MySQL группируются отдельно по типу: подключение, запрос, prepared statement, значение и прочие ошибки.

Слишком большие `.log`-файлы сжимаются для экономии места. Связанные с ними `.json`-данные группировки удаляются при сжатии, потому что слишком большие логи считаются неактивными. Логи автоматически не удаляются.
