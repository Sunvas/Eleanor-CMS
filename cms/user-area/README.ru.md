[English](README.md)

В этом каталоге находятся шаблоны публичной пользовательской части.

Пользовательская часть содержит только несколько общих системных шаблонов. Файл `maintenance.php` используется, когда сайт закрыт на обслуживание, а `error.php` — для вывода страниц ошибок. Содержимое запрошенных страниц выводят юниты, и каждый юнит сам решает, какие шаблоны использовать и где их хранить.

Рекомендуемая структура:

* `index.php` — основной костяк страницы, используемый шаблонами пользовательской части.
* `includes/` — переиспользуемые фрагменты шаблонов, подключаемые другими шаблонами.
* `l10n/` — файлы локализации общих шаблонов пользовательской части.
* `Widget*.php` — шаблоны виджетов для мест, куда владелец сайта может вставить произвольное содержимое: рекламу, баннеры, уведомления или другие динамические блоки.
* `unit_name/` — рекомендуемое место для шаблонов конкретного юнита.
