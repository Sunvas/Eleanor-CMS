[English](README.md)

В этом каталоге находятся файлы локализации, используемые общими шаблонами пользовательской части.

Имена файлов должны соответствовать одному из форматов:

- `[name]-[code].php` — группа локализации с кодом языка.
- `[code].php` — общий файл локализации без имени группы.

Каждый файл должен возвращать массив. Значения могут быть строками или Closure для динамически формируемого локализованного текста.
