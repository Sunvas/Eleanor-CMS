[English](README.md)

В этом каталоге находятся необязательные файлы виджетов, используемые для обработки произвольного содержимого перед вставкой в заранее предусмотренные места публичной пользовательской части.

Места размещения виджетов и их визуальная разметка определяются frontend-шаблонами. Каждое место может настраиваться из панели администратора: у него есть название, описание, необязательный файл виджета и необязательное содержимое, введённое владельцем сайта.

Если файл виджета не выбран, содержимое виджета может быть вставлено напрямую в место размещения. Если файл выбран, ему передаётся переменная `$content`, а файл возвращает содержимое, подготовленное для вставки. Разметка места размещения на странице остаётся под управлением frontend-шаблона.
