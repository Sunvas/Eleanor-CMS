[English](README.md)

В этом каталоге находятся публичные статические файлы, отдаваемые web-сервером напрямую: CSS, JavaScript, изображения, иконки, шрифты, загруженные файлы и похожие ресурсы.

Интерпретация PHP-скриптов в этом каталоге должна быть отключена.

Рекомендуемая структура:

- `admin-panel/` — статические файлы панели администратора.
- `user-area/` — статические файлы публичной пользовательской части.
- `unit_name/` — статические файлы отдельного юнита.
- `uploads/` — файлы, загруженные через CMS.
- `avatars/` — аватары пользователей.
